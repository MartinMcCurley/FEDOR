# FEDOR - Poker Bot Project

## Overview
FEDOR is a poker bot project developed for educational purposes, focusing on analyzing and improving poker strategies using machine learning techniques.

## Project Objectives
- Implement robust game state recognition
- Develop intelligent decision-making algorithms
- Create a user-friendly interface for bot control and monitoring

## Technical Stack
- **Language:** Python
- **GUI Framework:** PyQt6
- **Image Processing:** OpenCV
- **Machine Learning:** TensorFlow, scikit-learn

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/YourUsername/FEDOR.git
   cd FEDOR
   ```

2. Create and activate virtual environment:
   ```bash
   python -m venv FEDORenv
   .\FEDORenv\Scripts\activate  # Windows
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
